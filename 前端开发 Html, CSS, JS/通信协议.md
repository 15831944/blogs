# 规程, Procedure

在数据通信的早期, 对通信所使用的各种规则都称为规程, 规程专指物理层协议

# 协议, Protocol

具有体系结构的计算机网络就开始使用协议这一名词替代规程

# 规约, Specification

规约是经过相互协议规定下来的共同遵守的条款, 含义上同协议
