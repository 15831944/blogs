# 贷款

1. 只需要身份证就可以确定贷款额度
2. 必须办理农行借记卡还款, 存入后到期自动扣费
3. 首付53600变成54600

# 购车合同

办完贷款才能签合同并验车

# 手续

1. 办居住证才能挂牌, 行驶证上车辆登记地址是居住证上的地址
2. 车辆识别代码即车架号, 要与合同上的核对

1. 记得购置税和保险要多退少补, 列出明细
2. 登录交管12123客户端查询信息是否正常
3. 居住证明归还吗？
