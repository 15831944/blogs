
# 查看当前可用的代码段 snippets

`alt-shift-S`

快捷键配置如下

```
'atom-text-editor:not([mini])':
  'alt-shift-S': 'snippets:available'
```

# 代码段汇总如下

| 按键 | 功能 | 含义 |
| :- | :- | :- |
| b | bond text | 粗体 |
| img | embedded image | 嵌入图片 |
| legal | Copyright | 版权信息 |
| i | ** | 斜体 |
| l | \[]() | 链接 |
| code | ``` | 代码片 |
| t | - [ ]  | todo 列表 |
| lorem | lorem ipsum | 乱数假文 |
