# UIView 控件

1. 大多数控件都可以通过这三种模式使用, 并且几乎都支持一种以上的模式, 可根据需求选择合适的模式
2. 所有 iOS 控件都是 UIControl 的子类, 都能够触发操作方法
3. 大多数控件都支持被动模式, 所有控件都支持静态或者不可见模式

# 控件模式

1. 动态: 按钮, 能够触发操作方法
2. 静态
3. 被动: 标签, 图像控件, 网页的文本框

# 控件状态

1. 普通(normal):
2. 突出显示(highlighted): 正在被使用, 例如按钮处于被按压状态
3. 禁用(disabled): 控件被关闭
4. 选中(selected): 已启用或被选中, 用户不再直接使用时可继续保持选中状态
