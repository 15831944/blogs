```
char c = ‘a’;
char *p = &c;
```

| 表达式 | 右值           | 左值    | 对p的副作用 | 对c的副作用 |
| -------- | -------------- | --- | --- | --- |
| p        | p的内容(c的地址) | p的地址 |  |  |
| &p       | p的地址         |        |  |  |
| *p       | c的内容         | c的地址 |  |  |
| *p + 1   | c的内容+1       |        |  |  |
| *(p + 1) | c后面位置的内容  | c后面位置的地址 |  |  |
| ++p      | c后面位置的地址  |              | c后面位置的地址 |  |
| p++      | c的地址         |              | c后面位置的地址 |  |
| *++p     | c后面位置的内容  | c后面位置的地址 | c后面位置的地址 |  |
| *p++     | c的内容         |  c的地址      | c后面位置的地址 |  |
| ++*p     | c的内容+1       |              |               | c的内容+1 |
| (*p)++   | c的内容         |              |               | c的内容+1 |
| ++*++p   | c后面位置的内容+1 |             | c后面位置的地址 | c后面位置的内容+1 |
| ++*p++   | c的内容+1       |              | c后面位置的地址 | c的内容+1 |
