# 2 string 类

## 2.1 string 对象初始化

```
string s1;			// 定义空的字符串对象
string s2("abc");	// 用字符串字面值初始化字符串对象
string s3(s2);		// 将 s3 初始化为 s2 的一个副本
string s4(n,'c');	// 将 s4 初始化为 n 个 字符 'c' 的副本
```

字符串拼接只适用于字符串对象, 必须时变量, 两个字面值字符串不可以使用 "+" 进行拼接
